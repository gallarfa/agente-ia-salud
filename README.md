# 🧠 Agente Inteligente de Clasificación de Urgencias en Salud Mental

## 📌 Descripción del Proyecto

Este proyecto consiste en un agente desarrollado en **Python** que analiza el motivo de consulta de un paciente dentro de un sistema de salud mental y clasifica automáticamente el nivel de urgencia del caso.

El sistema recibe una descripción textual del problema del paciente y asigna una prioridad:

- 🔴 **ALTA**
- 🟡 **MEDIA**
- 🟢 **BAJA**

El objetivo es asistir al personal médico o administrativo para priorizar rápidamente los casos más críticos.

---

# ❗ Problema que resuelve

En centros de salud mental muchas consultas llegan al mismo tiempo y no siempre existe un método rápido para determinar qué paciente necesita atención inmediata.

Este agente ayuda a:

- Reducir tiempos de clasificación
- Priorizar pacientes con situaciones críticas
- Mejorar la atención inicial
- Automatizar una tarea repetitiva
- Brindar apoyo al personal administrativo o profesional

### Ejemplo de situaciones:

**Prioridad ALTA**
- Intento de suicidio
- Crisis severa
- Ataques violentos
- Riesgo inmediato

**Prioridad MEDIA**
- Ansiedad fuerte
- Estrés laboral
- Crisis emocional moderada

**Prioridad BAJA**
- Consultas generales
- Terapia preventiva
- Seguimiento emocional
